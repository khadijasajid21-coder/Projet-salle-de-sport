   # Projet-salle-de-sport
# Présentation du Projet:
L'objectif est de concevoir une application de bureau (Desktop) permettant de centraliser la gestion des membres, de leurs contrats et du suivi financier d'une salle de sport. L'application vise à automatiser l'identification des membres en retard de paiement et à simplifier l'administration quotidienne.

# Conception Technique (Java NetBeans) :
L'application utilise l'architecture MVC (Modèle-Vue-Contrôleur) pour séparer la logique métier de l'interface utilisateur :
•	Interface (Swing) : Utilisation de JInternalFrame pour les fenêtres,  JTable pour afficher les listes d'abonnés et d'impayés et JFrame pour la classe Main.
•	NetBeans : Environnement de développement intégré (IDE) utilisé pour codé, concevoir les interfaces graphiques et gérer le projet.
•	Connexion BD : Utilisation du driver JDBC pour la communication entre Java et MySQL.
•	jFreeChart : pour générer des graphiques/statistiques (jfreechart-1.5.4.jar).
•	jCalendar : pour la sélection des dates dans l’interface (jcalendar-1.4.jar).
•	jCrypt      : pour le hachage des mots de passe (jbcrypt-0.4.jar).
•	JavaMail : pour l’envoi d’emails (mail-1.4.1.jar et activation-1.1.1.jar).
•	Drawio    : Utilisé pour la modélisation UML.
